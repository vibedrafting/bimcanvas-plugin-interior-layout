# 设计品质维度

> 好的设计不是先随便做、再检查有没有碰巧做对——好的设计从第一步就知道自己要往哪走。
> 这五个维度既是**策略阶段的设计目标**，也是**审查阶段的验证标准**。
> 有了"进门直达台盆"的目标，选墙时不需要公式；审查时只需验证"进门后首个使用点是台盆吗？"——目标驱动让每个决策都有判据。

---

## 两层评价

```
Layer 1  工程合规    "有没有错？"    几何合法 + 通行可达 + 功能完整
Layer 2  设计品质    "好不好？"      下文五个维度
```

每层是上一层的前提——有错误的布局谈不上品质。Layer 1 不通过时不进入 Layer 2。

**区分**：900mm 通道宽度是 Layer 1 底线（"能走过去"）；但一条 900mm 的通道如果需要折返再前进，Layer 2 的动线设计仍不合格（"走得不自然"）。

**截图复核**：`validate_layout` 通过只代表几何合法，不代表设计自然。截图审查时必须看当前视觉证据，尤其检查：
- 窗帘是否按合同完整到达两端墙面，是否因验证碰撞被截断。
- 衣柜是否无故偏小，门侧留白是否有明确用途。
- 卧室床体是否过度占压空间，导致收纳、窗帘或通道被动让步。
- 留白是否是入口缓冲/通道/操作净墙，而不是家具放完后的残留。

---

## 五个品质维度

### 动线设计

> 来自空间阅读·动线方向——人每天走进房间的路线决定了使用体验。
> **核心问题**：从入口到每个功能点的路径直觉吗？功能沿动线递进吗？

**声明时**：描述入门路径意图，用 `(动线)` 标签
- `进门右转直达台盆(动线)`
- `餐桌靠近入口缩短上菜动线(动线)`

**判断时**：
- ✓ 卫生间门偏西，洁具放东墙——进门自然右转直达台盆
- ✗ 卫生间门偏西，洁具放西墙——进门后要折返
- ✓ 卧室：更衣区在入口侧，床在深处——进入即到达使用点
- ✗ 卧室：床紧挨门口，入门即见床——无过渡，路径无逻辑

---

### 空间意图

> 来自 generate-zoning「分区赋予空间以意图」——没有意图的空间只是家具放完后的剩余。
> **核心问题**：每个区域有可命名的功能身份吗？有无意识的剩余空间吗？

**声明时**：命名区域的功能角色，用 `(空间意图)` 标签
- `L形延伸作为独立换衣区(空间意图)`
- `凹角赋予书桌工作区身份(空间意图)`

**判断时**：
- ✓ L形延伸做更衣区——异形空间被赋予明确功能
- ✗ L形延伸空置——"家具放完剩下的"
- ✓ 有意识的留白：入口缓冲区（过渡需要）、主通道（动线需要）
- ✗ 无意识的残留：<600mm 窄缝，既不是通道也不是收纳

---

### 功能叙事

> 来自 generate-zoning「家具属于各自区域的功能叙事」——连贯的故事胜过碎片堆砌。
> **核心问题**：同一区域的家具讲述连贯的功能故事吗？

**声明时**：描述家具群组形成的功能主题，用 `(功能叙事)` 标签
- `东墙统一洗漱+如厕功能(功能叙事)`
- `北墙L形衣柜构成连续存储叙事墙(功能叙事)`
- `电视柜+沙发+茶几形成完整休闲叙事组(功能叙事)`

**判断时**：
- ✓ 台盆+镜柜+毛巾架组成洗漱区——功能群组连贯
- ✗ 衣柜-空隙-床头柜-书桌混杂同一面墙——碎片化堆砌
- ✓ 连续衣柜墙——换衣视线所及全是收纳，存储叙事无断点
- ✗ 衣柜被梳妆台插断——存储叙事断裂，功能混杂

---

### 空间节奏

> 来自 generate-zoning「区域间的留白是有意识的空间节奏」——有意识的递进而非均匀填充。
> **核心问题**：入口到深处有隐私/密度的递进吗？留白是有意识的吗？

**声明时**：描述从入口到深处的功能序列，用 `(空间节奏)` 标签
- `洗手→如厕→淋浴由浅入深(空间节奏)`
- `入口→换衣区→床的隐私递进(空间节奏)`
- `餐厅(过渡)→客厅(核心)→窗景(深处)的渐进体验(空间节奏)`

**判断时**：
- ✓ 卫生间：台盆(公开)→马桶(半私密)→淋浴(最私密)——隐私梯度清晰
- ✗ 马桶紧挨门口——进门一览无余，隐私梯度倒置
- ✓ 卧室：更衣区在入口侧（过渡），床在深处（私密）
- ✓ 客餐厅：餐厅靠入口（过渡/半公共），客厅朝窗（休闲/深处）

---

### 采光通风

> 来自空间阅读·采光轴——窗户是室内最珍贵的资源，不应被浪费。
> **核心问题**：窗户资源被合理利用了吗？光路通畅吗？

**声明时**：描述功能与光线的匹配关系，用 `(采光通风)` 标签
- `梳妆台靠南窗采光(采光通风)`
- `南窗最优采光给主活动区(采光通风)`

**判断时**：
- ✓ 梳妆台/书桌靠窗——需要自然光的功能获得了采光
- ✗ 高衣柜正对窗——切断光线，深处空间变暗
- ✓ 客厅主活动区朝南窗——使用频率最高的区域得到最优采光
- ✓ 卫生间淋浴区临窗——通风排湿（此处临窗是正面示例，非违规）

## 维度选取参照

### 空间特征 → 优先维度

| 空间特征 | 优先选取 |
|---------|---------|
| 入口位置偏向一端（非居中） | 动线设计 |
| 明显的入口→深处轴向 | 空间节奏 |
| 异形区/L形延伸（≥3m²） | 空间意图 |
| 多件家具集中同一墙面 | 功能叙事 |
| 自然采光明显（大窗/朝南） | 采光通风 |

无显著特征时的默认组合：

| 房间类型 | 默认维度 |
|---------|---------|
| 卫生间 | 动线设计 + 空间节奏 |
| 卧室 | 空间节奏 + 功能叙事 |
| 客餐厅 | 功能叙事 + 空间节奏 + 采光通风 |

### 声明格式

```
**设计目标**：[目标描述(维度名)] / [目标描述(维度名)] / [目标描述(维度名，如有)]
```

示例（卫生间）：
`**设计目标**：进门右转直达台盆(动线) / 东墙统一洗漱+如厕功能(功能叙事) / 洗手→如厕→淋浴由浅入深(空间节奏)`

示例（主卧）：
`**设计目标**：L形衣柜构成连续存储叙事墙(功能叙事) / 入口→换衣区→床的隐私递进(空间节奏) / 梳妆台靠南窗采光(采光通风)`

示例（客厅）：
`**设计目标**：电视柜+沙发+茶几形成完整休闲叙事(功能叙事) / 餐厅→客厅→窗景的渐进体验(空间节奏) / 南窗采光最大化给主活动区(采光通风)`
