# 卫生间策略

适用：tags 含 `shower`/`toilet`/`washing` 的空间。卫生间是**模板匹配型空间**——测量空间 → 选择布局模式 → 参数化定位洁具。Agent 角色：工程师（精度优先，自由度极低）。

## 一、基础规则（所有模式适用）

- **【建议】排布方向**：入口侧 → 深处（台盆靠入口，淋浴在最里）——动线自然，隐私递进，干湿分离
- **【建议】台盆贴角**：台盆贴靠相邻墙角，禁止留 <600mm 无功能缝隙——窄缝积灰难清洁
- **【建议】坐便器居中**：先定义“可用如厕段”，再让坐便器中心线对齐该段中心线；可用如厕段 = 台盆远离入口侧边沿 → 淋浴屏/湿区边沿；无淋浴时 → 该墙面可用末端；计算时扣除门禁区、通道、墙体转折和不可用段
- **【建议】功能相邻≠边沿贴靠**：坐便器不是台盆的附属件，不从台盆边缘顺序拼接；“入口 → 台盆 → 坐便器 → 淋浴”表达功能顺序，不表达贴边排队。若可用如厕段过短，才允许靠近台盆侧退化布置，但仍需保留必要使用净距
- **【建议】台盆选型**：可用墙面 > 850mm 时选定制台盆(horizontal_fill)，最大化台面空间
- **【必须】窄卫生间规则**：宽度 ≤ 1800mm 时，洁具集中同侧墙面——分列两侧中间通道无法通行

WHY：马桶贴台盆会牺牲如厕两侧操作空间，也会让洗漱区和如厕区混在一起。用“中心线对齐可用如厕段中心线”替代“从台盆边缘顺序拼接”，可以把定位锚点从台盆边缘转移到可用墙段本身。

## 二、模式总览

| 模式 | 适用房型 | 核心特征 | 淋浴类型 |
|------|---------|---------|---------|
| **A: 单侧线性** | 窄长型，短边≤1800mm | 全部沿一侧墙，对侧留通道 | mod_shower_linear |
| **B: 淋浴占远端** | 短边>1800mm，长宽比>1.5 | 远端整面淋浴，近端灵活分布 | mod_shower_linear 或 mod_shower_corner |
| **C: 近方形分墙** | 近方形，面积≤6㎡ | 洁具分布相邻墙面，角落淋浴房 | mod_shower_corner |
| **D: 宽裕型** | 面积>6㎡ | 三面墙各放一类，可含浴缸 | mod_shower_corner |
| **E: 半卫** | 无淋浴需求 | 台盆+坐便器，最简配置 | 无 |

## 三、模式选择决策树

按顺序判断（命中即停）：

1. **有淋浴需求？** zone.tags 含 `shower`？
   - 否 → **模式E**
   - 是 → 继续
2. **短边 ≤ 1800mm？** rawBoundary 最小外接矩形短边
   - 是 → **模式A**
   - 否 → 继续
3. **长宽比 > 1.5？**
   - 是 → **模式B**
   - 否 → 继续
4. **面积 > 6㎡？**
   - 否 → **模式C**
   - 是 → **模式D**

**门位置修正**（从 boundarySegments 的 door 段确定门位置）：门在短边 → 台盆从门侧开始；门在长边 → 台盆靠近门所在长边的近端。

## 四、各模式详细定位

### 模式A：单侧线性（短边≤1800mm）

所有洁具沿一侧长墙排列，对侧保留完整通道(≥700mm)。
**墙面选择**：
- 门在长边 → 对面长墙
- 门在短边 → 入门后开阔侧的长墙
  判断方法：比较 door 段中点与短边中点——门偏向墙A → 入门后开阔侧在墙B → 洁具放墙B

WHY（→ 动线设计目标）：人从偏向一端的门进入窄空间，自然向开阔侧行走——洁具在行走方向上 = 进门直达，动线最短最自然。反过来放在门偏向的一侧，进门后要折返。

> 开阔侧原则适用于所有单侧布置模式（A/B）。双侧模式（C/D）的墙面分配由功能分组决定，不适用此规则。
**定位**（从门端到深处）：
- 台盆：贴靠门端墙角
- 坐便器：位于台盆远离入口侧边沿↔淋浴屏/湿区边沿的可用如厕段内，中心线对齐该段中心
- 淋浴屏：mod_shower_linear(horizontal_fill)，连接两面实墙，宽度=短边宽度

### 模式B：淋浴占远端（短边>1800mm，长宽比>1.5）

远端整面淋浴形成独立湿区，近端洁具分散于不同墙面。
**定位**：
- 淋浴：远端，mod_shower_linear 横跨短边 或 mod_shower_corner 占远端角落
- 台盆：入口侧角落，贴靠墙角
- 坐便器：与台盆不同墙面，在台盆与淋浴之间的区域

### 模式C：近方形分墙（面积≤6㎡）

洁具分布于相邻墙面，矩形淋浴房占远离门角落。
**定位**：
- 淋浴房：mod_shower_corner，远离门的角落
- 台盆：门侧墙面，贴靠墙角
- 坐便器：与淋浴房同侧或相邻墙面

### 模式D：宽裕型（面积>6㎡）

三面墙各放一类洁具，中央通道宽裕。可考虑浴缸替代/补充淋浴。
**定位**：
- 淋浴房：mod_shower_corner，门对面墙角落，尺寸可按空间放大
- 台盆：门一侧墙面，优先选定制台盆
- 坐便器：门另一侧墙面，单独占一段

### 模式E：半卫（无淋浴）

台盆+坐便器，最简配置。
**定位**：
- 台盆：门侧墙面，贴靠墙角
- 坐便器：位于台盆远离入口侧边沿至该墙面可用末端的可用如厕段内居中

### 模块选型速查

| 洁具 | 默认模块 | 条件触发替换 |
|------|---------|-------------|
| 台盆 | mod_basin_001 (fixed 850×550) | 可用墙面>850mm → mod_basin_custom_001 (horizontal_fill) |
| 坐便器 | mod_toilet_001 (fixed 400×650) | 无替换 |
| 淋浴 | mod_shower_linear (horizontal_fill) | 模式C/D → mod_shower_corner (parametric) |
| 花洒 | mod_shower_head (fixed 135×165) | 安装在淋浴区内实墙 |

## 五、避窗规则特例

卫生间的避窗规则与其他房间**相反**（design_principles 已标注此例外）：
- **淋浴区可临窗**——采光通风好，不受避窗规则限制
- **坐便器避免正对窗户**——隐私

## 六、策略推导示例

> 参照此密度和推理风格，用当前空间数据独立推导。注意设计目标——它是后续执行和审查的锚点。

**场景1**：1.54m × 2.54m 公卫，南墙入口偏右（偏东），北墙高窗

**空间识别**：
- 短边 1540mm ≤ 1800mm → 模式A（单侧线性）
- 门在南墙（短边）偏右（偏东）→ 入门后开阔侧在西 → 洁具沿西墙排列（进门自然左转即达）
- 北墙高窗 → 淋浴区可临窗

**策略推导**：
- 模式：A（单侧线性）
- 布置墙：西墙（2.54m）
- **设计目标**：进门左转直达台盆(动线) / 西墙统一洗漱+如厕功能(功能叙事) / 洗手→如厕→淋浴由浅入深(空间节奏)
- 台盆：西墙南端，贴靠南墙角，定制台盆(horizontal_fill)
- 坐便器：台盆北边沿至淋浴屏南边沿的可用如厕段内居中
- 淋浴屏：西墙北端，mod_shower_linear，宽度=1540mm
- 花洒：淋浴区内北墙（实墙）
- 通道：东侧保留 ≥700mm 完整通道

**场景2**：1.6m × 3.2m L形主卫，南墙入口偏左（偏西），北墙窗户

**空间识别**：
- 短边 1600mm ≤ 1800mm → 模式A（单侧线性）
- 门在南墙（短边）偏左（偏西）→ 入门后开阔侧在东 → 洁具沿东墙排列（进门自然右转即达）
- 北墙窗户 → 淋浴区可临窗

**策略推导**：
- 模式：A（单侧线性）
- 布置墙：东墙
- **设计目标**：进门右转直达台盆(动线) / 东墙统一洗漱+如厕功能(功能叙事) / L形转折天然分隔干湿(空间意图)
- 台盆：东墙南端，贴靠南墙角
- 坐便器：台盆北边沿至淋浴屏南边沿的可用如厕段内居中
- 淋浴屏：L形转折处，mod_shower_linear
- 花洒：淋浴区内

## 七、关键约束速查

**硬约束**：
- 窄卫(≤1800mm)洁具同侧 — 对侧通道不可压缩到无法通行
- 淋浴屏/房连接实墙(≥2面) — 结构支撑+防水可靠

**软指导**：
- 排布方向入口→深处 — 动线自然+隐私递进+干湿分离
- 台盆贴角 — 窄缝积灰
- 坐便器在可用如厕段居中 — 两侧对称操作空间
- 可用墙面>850mm时选定制台盆 — 最大化台面

**自由区域**：
- 洁具间的精确间距
- 配件位置（毛巾架、镜柜等）
